Project Overview

A spreadsheet-style tool to batch edit OrCAD schematic symbol properties with CSV/XML round-trip, validation, and naming normalization to keep BOM data consistent and error-free.

Designed to reduce tedious manual edits and speed up library maintenance for medium to large PCB projects.

Key Features (v1.0)

  • Spreadsheet-like grid for fast batch editing
  • CSV/XML import and export support
  • Property validation and normalization
  • Designed to reduce BOM mismatches and late-cycle edits
